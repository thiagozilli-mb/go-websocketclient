# Go WebSocket client

Simple implementation of a **client** for consumer the **mercadobitcoin** websocket system.

### Implementation
* [x] **gorilla/websocket** 

## How to use the client

```sh
./go run main.go --addr=ws.mercadobitcoin.com.br --path=ws
```
